# UltraSound Front End (UFE)

This repository consolidates multiple projects into a single monorepo for the Ultrasound Smart Switch system.

## Structure

- Upduino/
  FPGA firmware, UART interface, and hardware control logic

- Matlab/
  Simulation, signal processing, and GUI development

- SmartSwitch/
  Circuit design, KiCad schematics, and hardware implementation

## Notes

- Each subdirectory was originally an independent repository
- History has been preserved during import
- This repo is now the single source of truth going forward
